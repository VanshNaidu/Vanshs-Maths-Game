Vansh's Maths Game

Vansh's Maths Game is an educational product that was designed to assist people in learning basic arithmetic.

To start the game:

1. Run the program.
2. Choose between the operations: (1,2,3,4).
3. The more equations you get right the higher your score goes (+1 score for every correct answer).
4. Type 'exit' when desired to exit game.
5. Final score will be repeated at the end.
